    3. Create `src-tauri/src/ssh/manager.rs`:
       - `SshEvent` enum mirroring PtyEvent: Data { bytes: Vec<u8> }, Exit { code: Option<u32> }. Tagged serde.
       - `SshSession` struct: handle (russh::client::Handle<SshHandler>), channel_id (russh::ChannelId), reader_task (Option<JoinHandle<()>>).
       - `SshHandler` struct implementing russh::client::Handler trait -- minimal: accept all host keys (check_server_key returns true), implement data callback to forward to Channel.
       - `SshManager` struct: sessions (Arc<Mutex<HashMap<String, SshSession>>>), config_store (Arc<Mutex<SshConnectionStore>>).
       - `connect` method: load connection config by ID, create russh client with Config { inactivity_timeout: Some(Duration::from_secs(30)) }, authenticate (try key_path first with russh::keys::load_secret_key, then password if provided), open channel, request_pty("xterm-256color", cols, rows), request_shell, spawn tokio reader task that loops channel.wait() and sends SshEvent::Data/Exit to Tauri Channel. Store handle + channel_id (NOT the channel itself -- channel is consumed by reader).
       - `write` method: use handle.data(channel_id, data.into()) to write to remote.
       - `resize` method: use handle.window_change(channel_id, cols, rows, 0, 0) for remote PTY resize.
       - `disconnect` method: abort reader task, handle.disconnect(). Remove from sessions map.

       CRITICAL: The russh Channel is consumed by the reader task's wait() loop. Use the session Handle for write/resize operations, NOT the channel. Store channel_id separately.

    4. Create `src-tauri/src/ssh/commands.rs`:
       - `ssh_connect(connection_id, password: Option<String>, cols: u16, rows: u16, on_event: Channel<SshEvent>, ssh_state: State<SshManager>) -> Result<String, String>`: generates UUID session_id, calls manager.connect().

       ```typescript
       // --- SSH ---
       export type SshEvent =
         | { event: "data"; data: { bytes: number[] } }
         | { event: "exit"; data: { code: number | null } };

       export interface SshConnectionConfig {
         id: string;
         name: string;
         host: string;
         port: number;
         user: string;
         keyPath?: string;
         group?: string;
       }

       export interface SshGroup {
         name: string;
         connectionIds: string[];
       }

       export async function sshConnect(
         connectionId: string, password: string | null,
         cols: number, rows: number, onEvent: Channel<SshEvent>
       ): Promise<string> {
         return invoke<string>("ssh_connect", { connectionId, password, cols, rows, onEvent });
       }

       export async function sshWrite(sessionId: string, data: Uint8Array): Promise<void> {
         return invoke("ssh_write", { sessionId, data: Array.from(data) });
       }

       export async function sshResize(sessionId: string, cols: number, rows: number): Promise<void> {
         return invoke("ssh_resize", { sessionId, cols, rows });
       }

       export async function sshDisconnect(sessionId: string): Promise<void> {
         return invoke("ssh_disconnect", { sessionId });
       }

       export async function sshSaveConnections(connections: SshConnectionConfig[]): Promise<void> {
         return invoke("ssh_save_connections", { connectionsJson: JSON.stringify(connections) });
       }

       export async function sshLoadConnections(): Promise<SshConnectionConfig[]> {
         const raw = await invoke<string>("ssh_load_connections");
         return JSON.parse(raw);
       }
       ```
