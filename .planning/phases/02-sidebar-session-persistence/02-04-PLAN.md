   - `#[tauri::command] pub fn fs_create_file(path: String) -> Result<(), String>` -- creates empty file (and parent dirs if needed)
   - `#[tauri::command] pub fn fs_create_dir(path: String) -> Result<(), String>` -- creates directory (and parents)
   - `#[tauri::command] pub fn fs_rename(from: String, to: String) -> Result<(), String>` -- renames file or folder

2. Add IPC wrappers to `src/lib/ipc.ts`:
   ```typescript
   export async function fsCreateFile(path: string): Promise<void> {
     return invoke("fs_create_file", { path });
   }
   export async function fsCreateDir(path: string): Promise<void> {
     return invoke("fs_create_dir", { path });
   }
   export async function fsRename(from: string, to: string): Promise<void> {
     return invoke("fs_rename", { from, to });
   }
   export async function fsDelete(path: string): Promise<void> {
     return invoke("fs_delete", { path });
   }
   ```

3. Create `src/lib/fuzzyMatch.ts`:
   - Export `function fuzzyMatch(query: string, text: string): { match: boolean; score: number }`
   - Simple algorithm: Check if all query characters appear in text in order (case-insensitive)
   - Score: prefer matches at word boundaries, consecutive matches, and shorter strings
   - Export `function fuzzyFilter<T>(items: T[], query: string, getText: (item: T) => string): T[]`
   - Returns items sorted by score (best first), filtering out non-matches

4. Create `src/components/sidebar/ContextMenu.tsx`:
   - Props: `x: number, y: number, entry: FileEntry | null, parentPath: string, onClose: () => void, onRefresh: () => void`
   - Renders a positioned div at (x, y) with menu items:
     - If entry is a folder: "New File", "New Folder", "Rename", "Delete"
     - If entry is a file: "Rename", "Delete"
     - If entry is null (right-click on empty space): "New File", "New Folder"
   - "New File" / "New Folder": Shows inline input for name, then calls `fsCreateFile(parentPath + '/' + name)` or `fsCreateDir(...)`, then `onRefresh()`
   - "Rename": Shows inline input pre-filled with current name, then calls `fsRename(entry.path, parentPath + '/' + newName)`, then `onRefresh()`
   - "Delete": Shows confirmation ("Delete {name}?"), then calls `fsDelete(entry.path)`, then `onRefresh()`
   - Click-outside or Escape closes the menu
   - Style: dark background (var(--bg-secondary)), border (var(--border)), rounded corners, shadow
   - Each item: hover highlight, padding 6px 12px, font size 13px
