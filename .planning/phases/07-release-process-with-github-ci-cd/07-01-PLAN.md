  <action>
1. Add `tauri-plugin-updater = "2"` to `[dependencies]` in `src-tauri/Cargo.toml`.

2. In `src-tauri/tauri.conf.json`, add the updater plugin config inside the existing `"plugins"` object:
```json
"plugins": {
  "updater": {
    "endpoints": [
      "https://github.com/volodymyrsaakian/excalicode/releases/latest/download/latest.json"
    ],
    "dialog": false,
    "pubkey": ""
  }
}
```
Note: `dialog: false` because we handle the update prompt in the frontend. `pubkey` is empty string for unsigned builds (updater will detect versions but won't auto-install without signing). Adjust the GitHub owner/repo in the endpoint URL to match the actual repository.

3. In `src-tauri/capabilities/default.json`, add updater permissions to the `permissions` array:
```json
"updater:default",
"updater:allow-check",
"updater:allow-download-and-install"
```

4. In `src-tauri/src/lib.rs`, register the updater plugin by adding `.plugin(tauri_plugin_updater::Builder::new().build())` in the plugin chain (after the existing `.plugin()` calls, before `.manage()`).
