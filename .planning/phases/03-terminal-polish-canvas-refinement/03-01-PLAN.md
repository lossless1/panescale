2. In `TerminalNode.tsx`:
   - Import `SearchAddon` from `@xterm/addon-search` and `WebLinksAddon` from `@xterm/addon-web-links`
   - Add refs: `searchAddonRef = useRef<SearchAddon | null>(null)`, `searchOpenRef = useRef(false)`
   - Add state: `const [searchOpen, setSearchOpen] = useState(false)`, `const [searchQuery, setSearchQuery] = useState("")`, `const [processTitle, setProcessTitle] = useState("")`
   - In the terminal init useEffect, after `term.open(el)` and before the WebGL addon block:
     - Create SearchAddon: `const searchAddon = new SearchAddon(); term.loadAddon(searchAddon); searchAddonRef.current = searchAddon;`
     - Create WebLinksAddon: `const webLinksAddon = new WebLinksAddon((event, uri) => { event.preventDefault(); window.open(uri, '_blank'); }); term.loadAddon(webLinksAddon);`
     - Wire title change: `term.onTitleChange((title) => { setProcessTitle(title); });`
   - In cleanup, set `searchAddonRef.current = null`
   - Add Cmd/Ctrl+F handler in `handleKeyDown`: when key is 'f' or 'F' with mod key, `e.preventDefault(); e.stopPropagation(); setSearchOpen(true);`
   - Add search bar component inline (rendered above the terminal container div, inside the flex column):
     ```tsx
     {searchOpen && (
       <div className="nodrag nowheel nopan" style={{ display: 'flex', alignItems: 'center', gap: 4, padding: '4px 8px', background: 'var(--bg-titlebar)', borderBottom: '1px solid var(--border)', flexShrink: 0 }}>
         <input
           autoFocus
           value={searchQuery}
           onChange={(e) => { setSearchQuery(e.target.value); searchAddonRef.current?.findNext(e.target.value); }}
           onKeyDown={(e) => {
             e.stopPropagation();
             if (e.key === 'Escape') { setSearchOpen(false); setSearchQuery(''); searchAddonRef.current?.clearDecorations(); }
             if (e.key === 'Enter' && e.shiftKey) { searchAddonRef.current?.findPrevious(searchQuery); }
             else if (e.key === 'Enter') { searchAddonRef.current?.findNext(searchQuery); }
           }}
           placeholder="Search..."
           style={{ flex: 1, background: 'var(--bg-primary)', color: 'var(--text-primary)', border: '1px solid var(--border)', borderRadius: 4, padding: '2px 6px', fontSize: 12, outline: 'none' }}
         />
         <button onClick={() => { setSearchOpen(false); setSearchQuery(''); searchAddonRef.current?.clearDecorations(); }}
           style={{ background: 'none', border: 'none', color: 'var(--text-secondary)', cursor: 'pointer', fontSize: 12, padding: '0 4px' }}>
           &#x2715;
         </button>
       </div>
     )}
     ```

3. In `TerminalTitleBar.tsx`:

   - Replace the center span content: if `processTitle` is truthy, show `processTitle` instead of `truncateCwd(cwd)`. Show both by formatting as `processTitle - truncateCwd(cwd)` with the processTitle bold and the cwd in secondary color.

4. In `TerminalNode.tsx`, pass `processTitle={processTitle}` to `<TerminalTitleBar>`.
