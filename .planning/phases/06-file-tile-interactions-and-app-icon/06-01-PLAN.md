1. Create `src/hooks/useOpenTerminalFromTile.ts`:
   - Export a hook `useOpenTerminalFromTile(nodeId: string)` that returns a callback `(filePath: string | undefined) => void`
   - The callback:
     a. Gets the node's current position from `useCanvasStore` via `getState().nodes.find(n => n.id === nodeId)`
     b. Extracts parent directory from `filePath` using string split on `/` (drop last segment). If filePath is undefined/empty, use `useProjectStore.getState().activeProject()?.path ?? '~'` as fallback
     c. Computes spawn position: `{ x: node.position.x + (node.style?.width as number ?? 640) + 40, y: node.position.y }` (offset right of tile by tile width + 40px gap)
     d. Calls `useCanvasStore.getState().addTerminalNode(spawnPosition, parentDir)`
   - Use `useCallback` with proper deps. Import stores with `getState()` pattern inside callback (not reactive subscription) to avoid re-renders.

2. In `FilePreviewNode.tsx`:
   - Import `useOpenTerminalFromTile` from the new hook

   - Add `onDoubleClick={() => openTerminal(filePath)}` to the title bar div (the one with `className="drag-handle"`)
   - Add `cursor: "pointer"` style to title bar on hover via a simple title attribute: `title="Double-click to open terminal here"`

3. In `ImageNode.tsx`:
