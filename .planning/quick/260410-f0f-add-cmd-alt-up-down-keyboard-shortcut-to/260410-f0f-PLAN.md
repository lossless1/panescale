<objective>
Add a keyboard shortcut to navigate terminal piles in the sidebar from anywhere in the app. Pressing Cmd+Option+Up/Down on macOS (or Ctrl+Alt+Up/Down on Linux/Windows) moves the selection to the previous/next terminal in the pile list and focuses it on the canvas — same effect as clicking a pile row.
</objective>

<tasks>
- Extract the existing pile `onSelect` handler logic in `TerminalList.tsx` into a reusable `selectTerminal(nodeId)` callback.
- Attach a global `keydown` listener (capture phase) that:
  - Matches `Alt+(Cmd on macOS | Ctrl on Linux/Windows) + ArrowUp/ArrowDown`
  - Platform detection via `navigator.platform`
  - Wraps around (last ↓ → first, first ↑ → last)
  - Starts from `activeTerminalId` or `selectedNodeId`, falls back to first/last entry if none selected
  - Calls `preventDefault` + `stopPropagation` so we don't interfere with the terminal's own input
- Reuse `selectTerminal` in `renderNode.onSelect` so click + keyboard go through the same path.
</tasks>
